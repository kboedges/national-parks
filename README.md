# National Parks App

A quick app built using Vue during a guided tutorial, makes use of the NPS API https://www.nps.gov/subjects/developer/get-started.htm

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
